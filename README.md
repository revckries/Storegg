## 🌟 Overview

Welcome to **Storegg**, a one-of-a-kind mobile app that blends the excitement of a marketplace with the thrill of a minigame! Browse a wide range of products, purchase items with in-game coins, and even resell them later. Short on coins? Just head over to the minigame to try your luck and crack open some more! It's the perfect mix of shopping and fun, all in one place.

---

## 🚀 Key Features

* [cite_start]**Effortless Browsing:** View products in either a **list** or **grid** format to find what you're looking for[cite: 6, 10].
* [cite_start]**Search & Discover:** Quickly find specific items using the built-in search bar[cite: 11].
* [cite_start]**Unique Coin System:** Your **My Coins** balance lets you buy and sell products within the app[cite: 12].
* [cite_start]**The Minigame:** Play our egg-cracking minigame to earn random amounts of coins, with a limit of five tokens per session[cite: 24, 25].
* **Buy & Sell:** Purchase products with your coins and see them added to your **My Products** list. [cite_start]You can also sell items back to get your coins refunded[cite: 5, 37, 39, 41].

## 🚀 Get Started

Ready to dive in? Here's how you can get this project running on your local machine.

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation
1. **Clone this repository.**
2. **Install dependencies:**
   ```bash
   npm install
    ```
3.  **Start the app:**
    ```bash
    npx expo start
    ```

### Other Commands
To reset the starter code, run:
```bash
npm run reset-project
